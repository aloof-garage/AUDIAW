 77 | ## Design Philosophy
 78 | 
 79 | ### Core Principles
 80 | 
 81 | 1. **Performance First**: Audio processing requires real-time performance with minimal latency
 82 | 2. **Type Safety**: Leverage Rust's type system to prevent runtime errors in critical paths
 83 | 3. **Lock-Free Design**: Avoid locks in audio thread to prevent priority inversion
 84 | 4. **Separation of Concerns**: Clear boundaries between UI, business logic, and audio processing
 85 | 5. **Developer Experience**: Easy to understand, modify, and extend
 86 | 6. **Cross-Platform**: Single codebase for Windows, macOS, and Linux
 87 | 
 88 | ### Why This Architecture?
 89 | 
 90 | - **Rust for Audio**: Memory safety without garbage collection, zero-cost abstractions
 91 | - **React for UI**: Component-based architecture, rich ecosystem, fast development
 92 | - **Tauri Bridge**: Lightweight, secure, native performance
 93 | - **Monorepo**: Shared code, consistent versioning, simplified development

159 | ## Rust Crate Architecture
160 | 
161 | ### 1. audiaw-types
162 | 
163 | **Purpose**: Shared type definitions used across all crates
164 | 
165 | **Key Types**:
166 | ```rust
167 | // Core types
168 | pub type Sample = f32;              // Audio sample (-1.0 to 1.0)
169 | pub type SampleRate = u32;          // Hz (44100, 48000, etc.)
170 | pub type SamplePosition = u64;      // Position in samples
171 | pub type TrackId = u32;             // Unique track identifier
172 | pub type ClipId = u32;              // Unique clip identifier
173 | 
174 | // Transport state
175 | pub enum TransportState {
176 |     Stopped,
177 |     Playing,
178 |     Recording,
179 |     Paused,
180 | }
181 | 
182 | // Audio data structures
183 | pub struct AudioClip { ... }        // Audio clip with metadata
184 | pub struct Track { ... }            // Multi-clip track
185 | pub struct AudioConfig { ... }      // Engine configuration
186 | ```
187 | 
188 | **Design Decisions**:
189 | - Use `f32` for samples (industry standard, SIMD-friendly)
190 | - Use `u64` for positions (supports hours of audio at any sample rate)
191 | - Derive `Serialize`/`Deserialize` for IPC communication
192 | 
193 | ### 2. audiaw-engine
194 | 
195 | **Purpose**: Core audio processing engine
196 | 
197 | **Architecture**:
198 | ```rust
199 | pub struct AudioEngine {
200 |     state: Arc<ArcSwap<EngineState>>,  // Lock-free state
201 |     command_tx: Sender<EngineCommand>,  // Command channel
202 |     event_rx: Receiver<EngineEvent>,    // Event channel
203 | }
204 | 
205 | // Commands (UI → Engine)
206 | pub enum EngineCommand {
207 |     Play, Stop, Pause,
208 |     AddTrack(Track),
209 |     SetTrackVolume(TrackId, f32),
210 |     // ... 11 total commands
211 | }
212 | 
213 | // Events (Engine → UI)
214 | pub enum EngineEvent {
215 |     TransportStateChanged(TransportState),
216 |     PositionChanged(SamplePosition),
217 |     TrackAdded(TrackId),
218 |     // ...
219 | }
220 | ```
221 | 
222 | **Key Features**:
223 | - **Lock-Free State**: Uses `arc-swap` for atomic state updates
224 | - **Command Pattern**: Decouples UI from audio thread
225 | - **Event-Driven**: Async notifications to UI
226 | - **Thread-Safe**: Separate command processing and audio threads
227 | 
228 | **Processing Flow**:
229 | ```
230 | UI Thread          Command Thread       Audio Thread
231 |     │                    │                    │
232 |     ├─ Command ────────→ │                    │
233 |     │                    ├─ Process           │
234 |     │                    ├─ Update State ────→│
235 |     │                    │                    ├─ Read State
236 |     │                    │                    ├─ Process Audio
237 |     │                    │                    ├─ Output
238 |     │                    ├─ Event ───────────→│
239 |     ←─ Event ───────────┤                    │
240 | ```

461 | ## Audio Engine Design
462 | 
463 | ### Lock-Free Architecture
464 | 
465 | **Problem**: Locks in audio thread cause priority inversion and dropouts
466 | 
467 | **Solution**: Lock-free data structures
468 | 
469 | ```rust
470 | // Lock-free state updates
471 | pub struct AudioEngine {
472 |     state: Arc<ArcSwap<EngineState>>,  // Atomic pointer swap
473 | }
474 | 
475 | // Update state (command thread)
476 | state.rcu(|current| {
477 |     let mut new_state = (**current).clone();
478 |     new_state.volume = 0.8;
479 |     new_state
480 | });
481 | 
482 | // Read state (audio thread) - NO LOCKS!
483 | let state = self.state.load();
484 | let volume = state.volume;
485 | ```
486 | 
487 | ### Real-Time Safety
488 | 
489 | **Rules**:
490 | 1. No allocations in audio thread
491 | 2. No locks in audio thread
492 | 3. No system calls in audio thread
493 | 4. Bounded execution time
494 | 
495 | **Implementation**:
496 | - Pre-allocate all buffers
497 | - Use lock-free channels for commands
498 | - Process audio in fixed-size chunks
499 | - Use `arc-swap` for state updates

502 | 
503 | ```

### Data Flow

1. User drags audio file or clicks "Import"
2. Frontend calls `import_audio_file` Tauri command
3. Backend decodes file using symphonia
4. Backend resamples to project sample rate if needed
5. Backend generates waveform peaks (min/max per time window)
6. Backend creates AudioClip with data
7. Backend sends `EngineCommand::AddClip` to audio engine
8. Engine adds clip to track's clip list
9. Backend returns clip metadata + waveform to frontend
10. Frontend updates UI with new clip and waveform

| MP3 | symphonia | High | Lossy, ubiquitous |
| OGG | symphonia | Medium | Lossy, open format |

- **Background Decoding**: Use async for large files

```rust
        if let Some(stream) = self.input_stream.take() {

        Ok(std::mem::take(&mut self.recorded_samples))

        if self.is_recording {
            // Write to ring buffer
            for &sample in data {
                let _ = self.ring_buffer.push(sample);
            }
        }
    }

        while self.is_recording {
            // Read from ring buffer
            while let Some(sample) = self.ring_buffer.pop() {
                self.recorded_samples.push(sample);
            }
```

```rust
impl ParametricEq {
    pub fn new(sample_rate: f32) -> Self {

            frequency: 1000.0,
            gain: 0.0,
            q: 1.0,
            filter_type: FilterType::Peak,

                        band.frequency.hz(),
                        band.q,

                }
                FilterType::LowShelf => {
                    Coefficients::<f32>::from_params(
                        Type::LowShelf(band.gain),
                        self.sample_rate.hz(),
                        band.frequency.hz(),
                        band.q,
                    ).unwrap()
                }
                // ... other filter types
            };
            
            self.filters.push(DirectForm2Transposed::<f32>::new(coeffs));
        }
    }
}

        for ch in 0..channels {
            for i in (ch..samples.len()).step_by(channels) {
                let mut sample = samples[i];
                
                // Apply each filter in series
                for filter in &mut self.filters {
                    sample = filter.run(sample);

                samples[i] = sample;

        for filter in &mut self.filters {
            filter.reset_state();
        }
    }

        self.bypassed = bypass;
    }
    
    fn is_bypassed(&self) -> bool {
        self.bypassed
    }
    
    fn get_parameters(&self) -> EffectParameters {
        EffectParameters::Eq(EqParameters {
            bands: self.bands.clone(),
        })
    }
    
    fn set_parameters(&mut self, params: EffectParameters) {
        if let EffectParameters::Eq(eq_params) = params {
```

```rust
// crates/audiaw-effects/src/compressor.rs

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct CompressorParameters {
    pub threshold: f32,    // dB (-60 to 0)
    pub ratio: f32,        // 1:1 to 20:1
    pub attack: f32,       // ms (0.1 to 100)
    pub release: f32,      // ms (10 to 1000)
    pub knee: f32,         // dB (0 to 12)
    pub makeup_gain: f32,  // dB (0 to 24)
}

pub struct Compressor {
    params: CompressorParameters,
    envelope: f32,
    sample_rate: f32,
    bypassed: bool,
}

impl Compressor {
    pub fn new(sample_rate: f32) -> Self {
        Self {
            params: CompressorParameters {
                threshold: -20.0,
                ratio: 4.0,
                attack: 5.0,
                release: 50.0,
                knee: 3.0,
                makeup_gain: 0.0,
            },
            envelope: 0.0,
            sample_rate,
            bypassed: false,
        }
    }
    
    fn compute_gain_reduction(&self, input_db: f32) -> f32 {
        let threshold = self.params.threshold;
        let ratio = self.params.ratio;
        let knee = self.params.knee;
        
        if input_db < threshold - knee / 2.0 {
            // Below threshold
            0.0
        } else if input_db > threshold + knee / 2.0 {
            // Above threshold
            (threshold - input_db) + (input_db - threshold) / ratio
        } else {
            // In knee region (soft knee)
            let x = input_db - threshold + knee / 2.0;
            let y = x * x / (2.0 * knee);
            -y / ratio
        }
    }
}

impl AudioEffect for Compressor {
    fn process(&mut self, samples: &mut [f32], channels: usize) {
        if self.bypassed {
            return;
        }
        
        let attack_coeff = (-1.0 / (self.params.attack * 0.001 * self.sample_rate)).exp();
        let release_coeff = (-1.0 / (self.params.release * 0.001 * self.sample_rate)).exp();
        let makeup_gain_linear = 10.0_f32.powf(self.params.makeup_gain / 20.0);
        
        for i in (0..samples.len()).step_by(channels) {
            // Get peak level across channels
            let mut peak = 0.0_f32;
            for ch in 0..channels {
                peak = peak.max(samples[i + ch].abs());
            }
            
            // Convert to dB
            let input_db = 20.0 * peak.max(1e-6).log10();
            
            // Compute gain reduction
            let gain_reduction_db = self.compute_gain_reduction(input_db);
            let target_envelope = 10.0_f32.powf(gain_reduction_db / 20.0);
            
            // Envelope follower
            let coeff = if target_envelope < self.envelope {
                attack_coeff
            } else {
                release_coeff
            };
            self.envelope = target_envelope + coeff * (self.envelope - target_envelope);
            
            // Apply gain reduction + makeup gain
            let gain = self.envelope * makeup_gain_linear;
            for ch in 0..channels {
                samples[i + ch] *= gain;
            }
        }
    }
    
    fn reset(&mut self) {
        self.envelope = 0.0;
    }
    
    fn name(&self) -> &str {
        "Compressor"
    }
    
    fn set_bypass(&mut self, bypass: bool) {
        self.bypassed = bypass;
    }
    
    fn is_bypassed(&self) -> bool {
        self.bypassed
    }
    
    fn get_parameters(&self) -> EffectParameters {
        EffectParameters::Compressor(self.params.clone())
    }
    
    fn set_parameters(&mut self, params: EffectParameters) {
        if let EffectParameters::Compressor(comp_params) = params {
            self.params = comp_params;
        }
    }
}
```

```rust
// Freeverb algorithm implementation
pub struct Reverb {
    params: ReverbParameters,
    comb_filters: Vec<CombFilter>,
    allpass_filters: Vec<AllpassFilter>,
    sample_rate: f32,
    bypassed: bool,
}

struct CombFilter {
    buffer: Vec<f32>,
    index: usize,
    feedback: f32,
    damping: f32,
    filter_state: f32,
}

struct AllpassFilter {
    buffer: Vec<f32>,
    index: usize,
}

impl Reverb {
    pub fn new(sample_rate: f32) -> Self {
        // Freeverb tuning values
        let comb_tunings = [1116, 1188, 1277, 1356, 1422, 1491, 1557, 1617];
        let allpass_tunings = [556, 441, 341, 225];
        
        let mut comb_filters = Vec::new();
        for &tuning in &comb_tunings {
            let size = (tuning as f32 * sample_rate / 44100.0) as usize;
            comb_filters.push(CombFilter {
                buffer: vec![0.0; size],
                index: 0,
                feedback: 0.84,
                damping: 0.2,
                filter_state: 0.0,
            });
        }
        
        let mut allpass_filters = Vec::new();
        for &tuning in &allpass_tunings {
            let size = (tuning as f32 * sample_rate / 44100.0) as usize;
            allpass_filters.push(AllpassFilter {
                buffer: vec![0.0; size],
                index: 0,
            });
        }
        
        Self {
            params: ReverbParameters {
                room_size: 0.5,
                damping: 0.5,
                wet_level: 0.3,
                dry_level: 0.7,
                width: 1.0,
            },
            comb_filters,
            allpass_filters,
            sample_rate,
            bypassed: false,
        }
    }
}

impl AudioEffect for Reverb {
    fn process(&mut self, samples: &mut [f32], channels: usize) {
        if self.bypassed {
            return;
        }
        
        // Process stereo pairs
        for i in (0..samples.len()).step_by(channels) {
            let input_l = samples[i];
            let input_r = if channels > 1 { samples[i + 1] } else { input_l };
            
            // Mix to mono for reverb input
            let input = (input_l + input_r) * 0.5;
            
            // Process through comb filters
            let mut output = 0.0;
            for comb in &mut self.comb_filters {
                output += comb.process(input);
            }
            
            // Process through allpass filters
            for allpass in &mut self.allpass_filters {
                output = allpass.process(output);
            }
            
            // Mix wet/dry
            let wet = output * self.params.wet_level;
            samples[i] = input_l * self.params.dry_level + wet;
            if channels > 1 {
                samples[i + 1] = input_r * self.params.dry_level + wet * self.params.width;
            }
        }
    }
    
    fn reset(&mut self) {
        for comb in &mut self.comb_filters {
            comb.buffer.fill(0.0);
            comb.filter_state = 0.0;
        }
        for allpass in &mut self.allpass_filters {
            allpass.buffer.fill(0.0);
        }
    }
    
    fn name(&self) -> &str {
        "Reverb"
    }
    
    fn set_bypass(&mut self, bypass: bool) {
        self.bypassed = bypass;
    }
    
    fn is_bypassed(&self) -> bool {
        self.bypassed
    }
    
    fn get_parameters(&self) -> EffectParameters {
        EffectParameters::Reverb(self.params.clone())
    }
    
    fn set_parameters(&mut self, params: EffectParameters) {
        if let EffectParameters::Reverb(reverb_params) = params {
            self.params = reverb_params;
            // Update filter coefficients based on new parameters
        }
    }
}
```

        mut command: Box<dyn UndoableCommand>,
        state: &mut ProjectState,

        // Execute the command
        command.execute(state)?;
        
        // Add to history
        if let Some(transaction) = &mut self.current_transaction {
            // Part of a transaction
            transaction.push(command);
        } else {
            // Try to merge with last command
            if let Some(last) = self.undo_stack.last_mut() {
                if last.can_merge(&*command) {
                    last.merge(command)?;
                    return Ok(());

            self.undo_stack.push(command);

            self.redo_stack.clear();

                self.undo_stack.remove(0);
            }
        }
